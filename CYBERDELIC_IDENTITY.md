# Cyberdelic Identity

Cyberdelic is a generative tattoo language.
It combines cybernetic precision, tribal rhythm, and fractal mutation into visual systems that can be recomposed without limit.

## Vision

- Build a modular p5.js engine for cyber tribal tattoo drafting.
- Treat each motif as a rule system, not a fixed drawing.
- Produce outputs that feel ritualistic, mechanical, and alive.

## Aesthetic Principles

- Use tension between symmetry and asymmetry.
- Shape rhythm through repetition, interruption, and drift.
- Balance density and silence to preserve breathing space.
- Keep directional flow clear, as if each form is tracking a signal.
- Preserve a sharp silhouette that reads instantly on skin.

## Generative System Principles

- Compose final forms from small modules: `fractal`, `track`, `noise`, `mirror`, `warp`.
- Keep every module parameterized to allow broad variation spaces.
- Separate structure generation from rendering style.
- Use deterministic seeds for reproducible drafts.
- Prefer combinatorial pipelines over one-off sketches.

## Technical Principles

- Use p5.js as the real-time sketch and export surface.
- Keep modules pure when possible: input parameters in, geometry out.
- Define shared interfaces for reusable pattern operators.
- Build presets as data objects so styles can evolve without rewriting logic.
- Ensure each module can run alone or as part of a chain.

## Output Principles (Tattoo-Ready)

- Prioritize black-and-white contrast first, grayscale second.
- Protect line integrity at multiple scales.
- Avoid micro-detail that collapses during stencil transfer.
- Output with clean edges and high legibility on dark or light skin.
- Validate designs at both poster scale and body scale.

## Expansion Direction

- Preset library for mood families (cyber spine, biomech knot, signal halo).
- Layer composition system for center/core/ornament separation.
- Seed catalog for versioning and recall of strong drafts.
- Optional post-process filters for stencil and shading previews.

## Core Statement

Cyberdelic is not a single style sheet.
It is a programmable ritual grammar for endlessly recombinable cyber tribal tattoo forms.
